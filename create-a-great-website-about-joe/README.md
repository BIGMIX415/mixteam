# Mixon Cousins

A static fan tribute website about Joe Mixon and Jerry Mixon, their cousin connection, and their football achievements.

## Files

- `index.html` - page markup and sourced copy
- `styles.css` - responsive visual system
- `script.js` - smooth in-page navigation
- `assets/mixon-field.svg` - original editorial artwork

## Publish

This site is ready for GitHub Pages from the repository root. In GitHub, set Pages to deploy from the default branch and `/root`.
